GIT





git status
-mostra os arquivos modificados

git add
-adiciona mudanças feitas

git commit
-regisra as mudanças para o novo repositorio

git push
-envia as mudanças feitas para o repositorio

git pull
-inverso do git push, trazer as mudanças para o repositorio local / baixar commits novos

git log
-registro de commits

git remote
-mostra o origin, file origem

git revert (id do commit)
-reverte a mudança feita anteriormente

git reset --hard (id do commit que queremos voltar)
-desfazer o commit e a mudança no codigo

git commit --amend -m ""
-altera a mensagem do ultimo commit
